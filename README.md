## Project Description

Mumbai University SGPA Calculator (NEP 2020) is a modern web application built for Mumbai University Engineering students to calculate Semester SGPA accurately using the official credit structure.

It supports:

- Subject-wise marks entry
- Automatic grade calculation
- Grade Point calculation
- Earned Grade Points (EGP)
- SGPA calculation
- Subject-wise breakdown
- Quick Paste feature
- PDF Export
- Print Result
- Share Result
- Copy Result
- Responsive UI
- Dark Mode
- Mobile Friendly

## Features

- [x] Accurate SGPA calculation using the Mumbai University NEP 2020 grading scheme
- [x] Subject-wise marks input with inline validation
- [x] Automatic grade and grade point calculation
- [x] Earned Grade Points and final SGPA summary
- [x] Subject-wise breakdown table with credit-weighted results
- [x] Quick Paste parsing for faster data entry
- [x] Copy, share, print, and PDF export actions
- [x] Local save and recent calculations
- [x] Responsive dashboard layout
- [x] Dark mode and light mode support
- [x] Modern glassmorphism UI with smooth animations
- [x] Unit tests for grading, parsing, and SGPA logic
